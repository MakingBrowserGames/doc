# Sound

For the examples on this page, we'll assume the following
directory structure of a project:

~~~
project
 |- src
 |    \- Application.js
 \- resources
      \- sounds
         |- music
         |    \- levelmusic.mp3
         \- effect
              \- boink.mp3
~~~

## Example: Toggle Sound

~~~
m4_include(./examples/api/sound.js)
~~~
